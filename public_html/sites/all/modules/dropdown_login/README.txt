Description
-----------
This module creates a drop-down login available as a block.

Requirements
------------
Drupal 7.x

Installation
------------
1. Copy the entire dropdown_login directory the Drupal sites/all/modules/custom directory.

2. Login as an administrator. Enable the module in the "Administer" -> "Modules"

3. Add the new block "Dropdown Login" to the required region via "Administer" -> "Structure" -> "Blocks"
